# USB-Paranoiac
USB Security Scanner on Linux

<img width="128" height="128" alt="secure" src="https://github.com/user-attachments/assets/2aa66f19-7aca-4cc5-a3ba-3456683841b5" />

This program allows you to examine suspicious USB storage devices before plugging them into your Windows computer. This is not an antivirus; it does not have heuristic or automatic detection-cleanup features.
